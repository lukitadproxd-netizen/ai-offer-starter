# @ai-offer-starter/core

Core TypeScript helpers for designing ethical, sellable, and deliverable AI-assisted service offers.

## Functions

- `scoreOfferReadiness(input)` checks whether an offer is ready to sell.
- `generatePricingTiers(input)` creates beta, standard, and pro pricing tiers.
- `validateOffer(input)` reports errors and warnings for risky or unclear offers.
- `buildOutreachMessage(input)` creates a short ethical outreach message.
- `createDeliveryChecklist(input)` creates a delivery checklist for the service.
- `calculateSimpleUnitEconomics(input)` estimates setup revenue, monthly revenue, delivery hours, and hourly equivalent.

## Build

```bash
npm install
npm test
npm run build
```
