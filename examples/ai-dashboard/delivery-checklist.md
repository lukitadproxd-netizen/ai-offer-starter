# AI Dashboard Delivery Checklist

- [ ] Confirm decisions the dashboard should support.
- [ ] Define each metric and owner.
- [ ] Review safe sample data.
- [ ] Build first dashboard version.
- [ ] Check labels, units, and date ranges.
- [ ] Confirm no hidden private data is exposed.
- [ ] Write refresh instructions.
- [ ] Handoff dashboard and notes.
- [ ] Schedule first review session.
