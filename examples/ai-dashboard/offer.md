# AI Dashboard

## Customer

Small teams that track operations in spreadsheets but need a clearer recurring view.

## Problem

Important metrics are spread across manual sheets, screenshots, or ad hoc updates.

## Offer

A simple dashboard package:

- metric definition session
- spreadsheet cleanup recommendations
- dashboard layout
- manual refresh instructions
- monthly review checklist

## Outcome

The team has a clearer view of agreed metrics and a repeatable review process.

## Exclusions

- no paid API dependency
- no sensitive data collection by default
- no automated decisions
- no guaranteed performance improvement
