# Delivery Checklist

- [ ] Confirm customer, problem, outcome, scope, and exclusions.
- [ ] Confirm inputs needed and safe data boundaries.
- [ ] Create first version of deliverables.
- [ ] Review AI-generated outputs for accuracy and tone.
- [ ] Check that no guaranteed income or magic claims are included.
- [ ] Send draft for review.
- [ ] Apply agreed revisions.
- [ ] Package final files.
- [ ] Include usage notes.
- [ ] Confirm support window.
- [ ] Ask for feedback after use.
- [ ] Record improvements for the next version.
