# Contributing

Thanks for considering a contribution to `ai-offer-starter`.

This project helps people turn AI-assisted deliverables into clear, ethical, sellable, and deliverable service offers. Contributions should keep that standard.

## Good Contributions

- improve documentation clarity
- add realistic offer examples
- improve templates
- strengthen anti-scam checks
- add tests for existing core behavior
- improve JSON schemas
- fix broken links or typos

## What Not to Contribute

Please do not submit:

- guaranteed income claims
- passive income promises
- spam automation
- private data scraping workflows
- manipulative urgency patterns
- fake testimonial or review systems
- large app, frontend, or CLI features without prior discussion

## Local Setup

```bash
npm install
npm test
npm run build
npm audit
```

## Pull Request Checklist

- [ ] The change keeps the ethical philosophy intact.
- [ ] The change does not promise income or guaranteed outcomes.
- [ ] The change does not add spam automation.
- [ ] The change does not require paid APIs.
- [ ] Tests pass with `npm test`.
- [ ] Build passes with `npm run build`.
- [ ] Security audit is clean or documented with context.

## Example Contribution Standard

Every new example should include:

- clear customer
- clear problem
- practical outcome
- concrete deliverables
- exclusions
- validation or delivery notes
- no guaranteed revenue claims

## Reporting Problems

Use the bug report issue template for bugs. Use the idea template for new examples or documentation improvements.
