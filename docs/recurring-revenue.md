# Recurring Revenue

Recurring revenue should come from recurring value, not from hiding work behind vague retainers.

## Good Recurring Work

- monthly reporting
- content updates
- workflow maintenance
- dashboard review
- conversion notes
- customer feedback analysis
- small improvements

## Bad Recurring Work

- charging forever for a static file
- promising passive income
- pretending automation has no maintenance
- billing without a defined support or improvement scope

## Recurring Offer Template

Define:

- what happens every month
- how long it takes
- what the customer receives
- what is excluded
- how success is reviewed
- when to cancel or adjust the plan
