# What This Is Not

`ai-offer-starter` is not a shortcut around trust, skill, or delivery.

It is not:

- a money printer
- a get-rich-quick course
- a spam automation system
- a lead scraping workflow
- a promise of guaranteed revenue
- a replacement for customer discovery
- a replacement for human quality review
- a giant SaaS application

## Why This Matters

Many AI business ideas fail because they start with a tool and then search for someone to sell it to. This project starts from the opposite direction: find a real customer, understand a real problem, validate the need, and deliver a scoped service.

If an idea only works when you exaggerate results, hide risk, or send unwanted messages at scale, it does not belong here.
