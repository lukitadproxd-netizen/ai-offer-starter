# Roadmap

## Current Version

- Markdown docs, templates, and examples.
- Core TypeScript functions for readiness, pricing, validation, outreach, delivery, and unit economics.
- JSON schemas for offer, customer, validation, and delivery data.
- Minimal CI workflow.

## Near-Term Ideas

- Add CLI commands for scoring and template generation.
- Add more example offers from different markets.
- Add schema validation examples using local tools.
- Add contribution checklist for ethical examples.
- Add translations.

## Later Ideas

- Static documentation site.
- Local-only offer workbook generator.
- Community-maintained case studies.
- Optional integrations that do not require paid APIs.

## Non-Goals

- spam automation
- income guarantee logic
- paid API dependency
- giant SaaS surface area
